## Basic Commands

### Start Interactive Mode

```bash
neuron
```

This opens the interactive TUI where you can chat with the AI.

### Single Command

```bash
neuron "Add a dark mode toggle to the navbar"
```

### With Context

```bash
neuron --file src/App.tsx "Refactor this component"
```

## Command Options

| Option           | Description                         |
| ---------------- | ----------------------------------- |
| `--file, -f`     | Specify files to include as context |
| `--model, -m`    | Override the default model          |
| `--provider, -p` | Override the default provider       |
| `--verbose, -v`  | Show verbose output                 |
| `--dry-run`      | Preview changes without applying    |

## Examples

### Code Review

```bash
neuron --file src/utils.ts "Review this code for bugs"
```

### Generate Tests

```bash
neuron --file src/api.ts "Generate unit tests for this module"
```

### Refactor Code

```bash
neuron "Find all components using class components and convert them to functional"
```
