## Configuration File

neuron uses a `.neuron.yaml` configuration file in your project root.

```yaml
# .neuron.yaml
provider: openai
model: gpt-4

# Optional settings
temperature: 0.7
max_tokens: 4096
```

## Environment Variables

You can also configure neuron using environment variables:

```bash
export OPENAI_API_KEY="your-api-key"
export NEURON_MODEL="gpt-4"
```

## Provider Configuration

### OpenAI

```yaml
provider: openai
model: gpt-4
api_key: ${OPENAI_API_KEY}
```

### Anthropic

```yaml
provider: anthropic
model: claude-3-opus
api_key: ${ANTHROPIC_API_KEY}
```

### Local Models

```yaml
provider: ollama
model: llama2
base_url: http://localhost:11434
```
